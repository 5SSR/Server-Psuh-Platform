# Server-Psuh-Platform

一个由 **ChatGPT 辅助规划与推动开发** 的 IDC 服务器二手交易市场项目，目标是挑战式完成一个从产品设计、业务流程、功能规划到落地开发都相对完整的中介担保平台。

本项目聚焦 **VPS、独立服务器、NAT、云服务器、线路机等二手交易场景**，围绕卖家发布、买家下单、平台担保、资金托管、交付验机、纠纷处理、后台风控等环节，构建一个完整的交易闭环。

> 当前 README 已按开发计划重构，作为项目研发、协作、上线推进的统一说明文档。

## 1. 项目定位

`Server-Psuh-Platform` 是一个面向 IDC 二手服务器交易的担保型平台，核心目标是解决传统二手交易中常见的以下问题：

- 信息不透明：配置、到期时间、续费价格、线路真假难以确认
- 风险高：账号找回、虚假机器、黑历史机器、滥用机器、封禁机器
- 交易无保障：直接转账后卖家不交付，或交付与描述不符
- 售后难：出现纠纷后缺少统一仲裁与证据链

平台采用 **中介担保交易模式**：

1. 卖家提交机器信息并上架
2. 买家浏览并下单付款到平台
3. 卖家完成账号或控制权交付
4. 买家验机确认
5. 平台结算放款给卖家

---

## 2. 核心业务模式

### 2.1 担保交易
买家先付款到平台，平台托管资金，待机器交付和验机完成后放款给卖家。

### 2.2 寄售模式
卖家提前把账号或控制权交由平台保管，平台审核后上架，买家下单后由平台直接交付。

### 2.3 求购模式（规划中）
买家发布配置与预算需求，卖家提交符合条件的机器供买家选择。
